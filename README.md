Gdb Script for Ruby
===================
Helper scripts written by gdb script.
 
Install
-------
You need python-enable gdb, ruby builded debug mode.
```shell
git clone https://github.com/mathfur/gdb_scripts_for_ruby.git
```
```shell
./bin/notice_load foo.rb
```
